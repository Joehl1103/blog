# Blog Template Plan

## Goal

Create a basic blog website structure in this empty repository that is easy to open locally, easy to customize, and clear enough for future expansion into a fuller portfolio blog.

## Approach

1. Build a static homepage with:
   - a hero section
   - featured post area
   - latest posts grid
   - newsletter-style callout
2. Add a reusable article page structure for sample blog posts.
3. Keep the stack dependency-free:
   - `index.html`
   - `styles.css`
   - `app.js`
4. Include starter content and small JavaScript helpers so the template feels alive without introducing a framework too early.
5. Add core repo files:
   - `.gitignore`
   - `README.md`
   - `CLAUDE.md`
6. Run lightweight validation on the generated files.

## Notes

- Work directly on `main` in this repo unless the user says otherwise.
- Favor a clean, understandable structure over premature complexity.
