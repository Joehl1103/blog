# Project Context

## Overview

This repository is a static portfolio blog starter.

Current stack:

- HTML
- CSS
- Vanilla JavaScript

The project is intentionally simple so the structure is easy to understand and extend.

## Projects and Tasks

Current completed task:

- Initial blog website structure with homepage, sample posts, styles, and validation

Likely next tasks:

- Replace sample copy with real content
- Add images, author bio, and social links
- Decide whether to keep the site static or migrate to a framework later

## Conventions

- Unless the user says otherwise, work directly on `main` in this repository
- Keep the structure beginner-friendly and easy to reason about
- Prefer small reusable functions and straightforward page anatomy
